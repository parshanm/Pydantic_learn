In this file I'll show you diffrence between lax and strict mode in pydantic.

### 03-pydantic-extrafields
In this file we showed how to get and manage extra fields using config dict.

### 04-validating-in-high-level

Here we learn to how validate default values and assignment values in model level.

Here we learn how to make a model that can't be change anyway.

This file teachs you to make input data standard for your model like deleting white spaces or make strings lower or upper case.
